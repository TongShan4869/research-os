# Integrate Source

Use this workflow only after the user confirms one queued wiki inbox item.

1. Resolve the source through `registries/sources.yaml`, `registries/files.yaml`, and provider metadata.
2. Choose the source-type adapter: academic paper, code, article, dataset, figure, manuscript, or output.
3. Read only the minimum source material needed for the requested integration.
4. Update the source note and relevant wiki pages by section, not wholesale rewrite.
5. Update `index.md` with any new or changed page summaries.
6. Append a dated entry to `log.md`.

Academic papers: Zotero remains read-only. PDF/full text reading requires explicit confirmation.
