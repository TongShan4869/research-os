# Wiki Index

This is the LLM-maintained catalog for the Research OS wiki. Read this before opening random folders.

Research OS has three layers:

- Raw sources are the source of truth and are not rewritten.
- Registries are the machine-readable control plane.
- This wiki is the mutable synthesis layer that compounds over time.

## Start Here

- `Home.md`: dashboard and folder guide.
- `log.md`: chronological record of wiki maintenance.
- `wiki/inbox.md`: sources queued for explicit integration.
- `Synthesis/`: best first stop for project or topic understanding.
- `Sources/`: source notes linked to Zotero, local folders, and raw files.

## Page Types

| Folder | Use it for |
| --- | --- |
| `Projects/` | Project goals, status, linked sources, and open questions. |
| `Sources/` | Notes for papers, collections, articles, files, and provider metadata. |
| `Synthesis/` | Evolving summaries that combine multiple sources. |
| `Concepts/` | Concept explainers and definitions. |
| `Entities/` | Named people, labs, tools, institutions, places, or organisms. |
| `Claims/` | Evidence-backed statements, contradictions, and disputed findings. |
| `Methods/` | Protocols, models, measures, analyses, and workflows. |
| `Datasets/` | Dataset provenance, schema, paths, and usage notes. |
| `Results/` | Findings, figures, tables, outputs, and interpretations. |
