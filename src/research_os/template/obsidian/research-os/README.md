# Starter Vault

This folder can be copied or opened as a minimal Obsidian vault.

Research OS keeps a small LLM-maintained wiki core here:

- `index.md` catalogs maintained wiki pages.
- `log.md` records append-only wiki maintenance events.
- `wiki/inbox.md` queues sources for explicit integration.
- `Synthesis/`, `Entities/`, `Concepts/`, `Claims/`, `Methods/`, `Datasets/`, and `Results/` hold durable synthesis pages.
